Site Plan

Site Name

Azeroth Archives - I chose this name because the website will serve as a historical and technical library for players seeking knowledge about the world of Azeroth.

Site Purpose

The purpose of this site is to provide a comprehensive database for World of Warcraft expansions and a dedicated platform for guild recruitment, helping players find the right community and stay updated on game content.

Scenarios

  • Which is the best class to start playing?
  • Where can I find information about guild recruitment?
